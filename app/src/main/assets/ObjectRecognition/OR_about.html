The Object Recognition sample shows how to recognize and track an object.

Key Functionality

  • Recognize and track up to 2 objects simultaneously
  • Activate Extended Tracking
  • Manage camera functions: flash

Instructions

  • Scan an object with the Vuforia Object Scanner
  • Create a device database using the Target Manager
  • Compile the sample with your database
  • Point at an object to see an augmented box around it
  • Single tap to focus
  • Double tap to access the options menu
